🔴 BUG #1: render.yaml — Comando START incorreto

   ├─ Erro: startCommand: node server.js
   ├─ Causa: Build cria dist/index.js, mas Render tentava rodar server.js
   ├─ Impacto: Dashboard falha silenciosamente no Render

       ✅ APLICADA

🟡 BUG #2: dashboard/server.js — Falta debug logging
   ├─ Localização: dashboard/server.js

   └─ Solução: Adicionar 4 lines de debug output
