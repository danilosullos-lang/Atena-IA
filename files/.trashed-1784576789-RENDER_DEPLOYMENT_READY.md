## ⚠️ DEPENDÊNCIAS FALTANDO

Antes do deploy, instale:

```bash
# Localmente (para testar)

# No Render: já está no requirements.txt
```

Verificar se está em `requirements.txt`:
```bash
grep -E "aiohttp|aiofiles" requirements.txt
```

Se não estiver, adicione:
```
aiohttp>=3.9.0,<4.0
aiofiles>=23.2.1,<25.0
```

## 🚨 TROUBLESHOOTING

### Dashboard não abre no Render
```
Cause: startCommand ainda é "node server.js"
Fix: Verificar render.yaml - deve ser "node dist/index.js"
Arquivo: render.yaml linha 13
```

### API startup falha com "ModuleNotFoundError"
```
Cause: Dependência faltando
Fix: Adicionar em requirements.txt:
  - aiohttp
  - aiofiles
  - Quaisquer outras que faltem
```

### Skills/Missions não carregam
```
Cause: Diretórios skills/ ou modules/ não existem
Fix: ESPERADO! Os loaders procuram por arquivos, se não acham, continua
Status: Graceful degradation - sistema funciona sem eles
```

### Node server port error
```
Cause: PORT não é lido do env
Fix: Verificar dashboard/server.js - deve usar process.env.PORT || 3000
Status: FIXADO na versão atual
```
