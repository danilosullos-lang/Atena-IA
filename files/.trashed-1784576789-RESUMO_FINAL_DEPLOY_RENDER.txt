█ SE ALGO DER ERRADO
─────────────────────────────────────────────────────────────────────────────

Problema: "Dashboard não abre"
Causa: startCommand ainda é "node server.js"
Solução: 
  git checkout render.yaml
  (ou editar manualmente render.yaml linha 13 para "node dist/index.js")

Problema: "API com erro de import"
Causa: aiohttp ou aiofiles faltando
Solução: 
  grep aiohttp requirements.txt || echo "aiohttp>=3.9.0" >> requirements.txt
  grep aiofiles requirements.txt || echo "aiofiles>=23.2.1" >> requirements.txt

Problema: "Skills/Missions não carregam"
Causa: ESPERADO - directories vazios
Solução: Não é problema, sistema continua funcionando sem eles
